## Local development: using a secure OpenAI proxy

To avoid committing your OpenAI API key, run the small proxy server in `server/` which reads `OPENAI_API_KEY` from the environment.

PowerShell quick-start (from repository root):

1. Change directory and install dependencies:

    cd server
    npm install

2. Temporarily set the API key for the current PowerShell session:

    $env:OPENAI_API_KEY = "sk-..."

3. Start the proxy:

    npm start

4. In your client, send OpenAI requests to `http://localhost:3000/v1/chat/completions` instead of the public OpenAI URL.

Keep your real key out of source control ‚Äî do not paste it into files tracked by git.
